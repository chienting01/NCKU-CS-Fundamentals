# Exercise 02: Store Checkout

## 題目目標
練習使用條件判斷與運算子完成結帳流程。

## 題目情境
建立商店結帳計算程式，輸入資料如下：

1. 商品原價總金額
2. 是否為會員（yes/no）
3. 是否使用折扣碼（yes/no）

## 規則
1. 會員打 9 折（原價乘上 0.9）。
2. 若使用折扣碼，再折 50 元。
3. 折扣後金額不得小於 0。
4. 運費判斷：
   - 折扣後金額 >= 1000，免運
   - 否則運費 60
5. 輸出：
   - 原價
   - 折扣後金額
   - 運費
   - 應付總額

## 必須使用
- 算術運算子：`+`, `-`, `*`
- 比較運算子：`>=`
- 指派運算子：`-=`（或 `+=`）

## 加分挑戰
依照應付總額印出等級：
- `>= 2000`：`VIP Order`
- `>= 1000`：`Standard Order`
- 其他：`Small Order`
