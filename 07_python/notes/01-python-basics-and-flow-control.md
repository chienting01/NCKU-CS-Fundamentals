# Python 基礎與流程控制

## 為什麼這一份筆記很重要

Python 看起來比 C 簡單，語法也比較精簡，所以初學時很容易產生一種錯覺：

> 好像看得懂，就等於學會了。

但實際上不是這樣。

如果沒有把最基本的內容打穩，後面一進到：

- 函數
- 容器型別
- 檔案處理
- NumPy
- Pandas
- Kaggle

就很容易變成只是在跟著範例抄，而不是自己真的理解程式在做什麼。

所以這一份筆記的目的不是列語法表，而是建立最基本的 Python 思維：

1. Python 程式怎麼執行
2. 資料怎麼存進變數
3. 程式怎麼做判斷
4. 程式怎麼重複做事
5. 初學者最容易錯在哪裡

---

## Python 是什麼

Python 是一種高階、通用、可讀性高的程式語言。  
它的特色是：

- 語法相對簡潔
- 可直接執行
- 適合初學者入門
- 很常用在資料分析、自動化、機器學習、網頁開發等領域

在這個專案裡，Python 的角色不是取代 C，而是負責另一條線：

- 作為資料處理工具
- 作為 Kaggle 與分析任務的主要語言
- 作為實作小工具、小腳本的高效率語言

所以學 Python 的目的，不是只學會 `print()` 和 `if`，而是要逐步建立之後進資料分析的能力。

---

## Python 程式如何執行

和 C 語言不同，Python 通常不需要先手動編譯成可執行檔。  
最常見的做法是直接在終端機執行：

```bash
python main.py
````

有些環境會使用：

```bash
python3 main.py
```

這代表你可以很快地做到：

1. 建立一個 `.py` 檔
2. 寫程式
3. 直接執行
4. 看到結果
5. 修改後再執行一次

這種快速回饋，是 Python 很適合入門與寫工具的原因之一。

---

## `print()`：最基本的輸出

`print()` 是 Python 中最基本、也最常用的函式之一。
它的作用是把內容顯示到終端機。

例如：

```python
print("Hello, Python!")
```

執行後，畫面會看到：

```text
Hello, Python!
```

你可以把 `print()` 想成：

> 把程式裡的某個值或某段文字，顯示出來給自己看。

這在初學階段非常重要，因為你會很常用它來：

* 確認變數值
* 檢查條件判斷有沒有進去
* 看迴圈有沒有正常跑
* 驗證程式結果

---

## `input()`：接收使用者輸入

如果 `print()` 是輸出，那 `input()` 就是輸入。

例如：

```python
name = input("請輸入你的名字：")
print("你好，" + name)
```

這代表：

1. 程式先顯示提示文字
2. 等待使用者輸入
3. 把輸入內容存到 `name`
4. 再把 `name` 印出來

這是互動式程式最基本的形式。

### 要注意的一點

`input()` 預設讀進來的是**字串（str）**。
也就是說，即使你輸入的是數字，Python 一開始也會把它當成文字。

例如：

```python
age = input("請輸入年齡：")
print(age)
```

這裡的 `age` 預設是字串，不是整數。

如果你要把它當成整數使用，通常要轉型：

```python
age = int(input("請輸入年齡："))
```

這一點是初學者很常出錯的地方。

## 變數：用名字保存資料

變數可以理解成：

> 幫一份資料取一個名字，方便之後再拿來用。

例如：

```python
age = 20
name = "Alice"
height = 172.5
is_student = True
```

這裡代表：

* `age` 存的是整數
* `name` 存的是字串
* `height` 存的是浮點數
* `is_student` 存的是布林值

### 變數的意義

變數的重要性在於：
程式不可能每次都直接寫死資料，很多時候需要先把資料存起來，之後再：

* 印出來
* 做運算
* 拿來判斷
* 拿來更新

所以變數是程式中最基本的資料承載單位。

## 基本資料型態

Python 初學階段，先熟悉四種最基本型態就夠了。

### 1. `int`

整數，例如：

```python
count = 10
```

### 2. `float`

浮點數，也就是小數，例如：

```python
height = 172.5
```

### 3. `str`

字串，也就是文字，例如：

```python
name = "Python"
```

### 4. `bool`

布林值，只有兩種可能：

* `True`
* `False`

例如：

```python
is_passed = True
```

### 為什麼型態重要

雖然 Python 不像 C 那樣需要你先手動寫型別宣告，但型態仍然存在。
它會影響：

* 你能不能做某種運算
* 運算結果長什麼樣
* 某些函式怎麼處理資料

所以即使 Python 比較彈性，也不能忽略型態觀念。

---

## 運算子

運算子就是用來操作資料的符號。

---

### 算術運算子

最常見的有：

* `+` 加法
* `-` 減法
* `*` 乘法
* `/` 除法
* `//` 整數除法
* `%` 取餘數
* `**` 次方

例如：

```python
a = 10
b = 3

print(a + b)   # 13
print(a / b)   # 3.333...
print(a // b)  # 3
print(a % b)   # 1
print(a ** 2)  # 100
```

### 初學者要注意

`/` 和 `//` 不一樣：

* `/` 是一般除法
* `//` 是整數除法，會把小數部分去掉

---

### 比較運算子

比較運算子用來判斷兩個值之間的關係，結果通常是 `True` 或 `False`。

常見有：

* `==`
* `!=`
* `>`
* `<`
* `>=`
* `<=`

例如：

```python
score = 80
print(score >= 60)   # True
print(score == 100)  # False
```

---

### 邏輯運算子

邏輯運算子常用在條件組合中。

* `and`
* `or`
* `not`

例如：

```python
age = 20
is_student = True

print(age >= 18 and is_student)  # True
```

這代表條件是可以組合的，不是每次只能判斷一件事。

---

### 指派運算子

最基本的是：

```python
x = 5
```

也有一些簡寫方式，例如：

```python
x += 1
x -= 2
x *= 3
```

這些在迴圈與計數中很常用。

---

## 條件判斷：讓程式根據情況做不同事

現實世界中，很多事情不是固定流程，而是要看條件。
例如：

* 成績高於 60 才算及格
* 年齡大於等於 18 才算成年人
* 輸入錯誤時要提示重新輸入

這時候就需要條件判斷。

Python 使用：

* `if`
* `elif`
* `else`

來做條件分支。

例如：

```python
score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")
```

這段程式的意思是：

1. 如果分數大於等於 90，印出 A
2. 否則如果分數大於等於 80，印出 B
3. 其他情況印出 C

### 條件判斷的核心

不是背語法，而是理解：

> 程式可以根據不同情況，走不同路徑。

---

## Python 很重視縮排

這是 Python 初學者一定要記住的重點。

在 C 語言中，常常用 `{}` 表示程式區塊。
但 Python 不是這樣，它用**縮排**來表示某一段程式屬於哪個區塊。

例如：

```python
score = 85

if score >= 80:
    print("通過")
    print("表現不錯")
```

這裡兩行 `print()` 都屬於 `if` 區塊，因為它們有相同縮排。

如果縮排亂掉，程式就可能報錯或邏輯錯誤。

### 你可以把縮排理解成：

> Python 用排版來表示程式結構。

所以 Python 不是「好像格式漂亮一點比較好」，而是：

> 縮排本身就是語法的一部分。

---

## 迴圈：讓程式重複做事

很多程式工作都需要重複執行，例如：

* 印出 1 到 10
* 把資料列表逐一處理
* 反覆要求輸入直到輸入正確

如果沒有迴圈，就要一直重複寫很多相似程式碼。
所以迴圈的作用是：

> 讓程式自動重複執行某段邏輯。

Python 常見的迴圈有兩種：

* `for`
* `while`

---

## `for` 迴圈

`for` 通常用在你大致知道要重複幾次，或要逐一走過一組資料時。

例如：

```python
for i in range(5):
    print(i)
```

這會印出：

```text
0
1
2
3
4
```

### `range(5)` 的意思

它會產生一組從 0 到 4 的數字。
所以這個迴圈會跑 5 次。

### `for` 的使用情境

初學階段可以先把 `for` 想成：

> 按照某個範圍，一次一次往下做。

---

## `while` 迴圈

`while` 通常用在「只要條件還成立，就持續做」的情況。

例如：

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

這段程式代表：

* 只要 `count < 5`
* 就一直印出 `count`
* 每次加 1
* 直到條件不成立為止

### `while` 的使用情境

你可以把它理解成：

> 條件成立就一直做，條件不成立才停止。

---

## `break` 和 `continue`

這兩個常常會跟迴圈一起出現。

---

### `break`

`break` 會直接跳出整個迴圈。

例如：

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

這裡當 `i == 5` 時，整個迴圈直接停止。

---

### `continue`

`continue` 不會結束整個迴圈，而是跳過這一輪剩下的部分，直接進入下一輪。

例如：

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

這裡 `i == 2` 那一輪不會印出來，但迴圈還會繼續跑下去。

---

## Python 和 C 的初步差異

你目前同時在學 C，所以先建立一個基本對照觀念很有幫助。

### C 的特徵

* 型別需要明確宣告
* 需要編譯
* 比較接近底層
* 語法相對嚴格

### Python 的特徵

* 型別不需要手動先宣告
* 程式可直接執行
* 語法較簡潔
* 適合快速寫腳本與做資料處理

### 這代表什麼

不是哪個比較高級，而是用途不同。

* C 幫你建立基礎、結構、底層理解
* Python 幫你快速實作、處理資料、進行分析

所以兩者是互補，不是互斥。

## 初學者最常犯的錯

### 1. 忘記縮排

Python 對縮排很敏感。
一旦縮排錯誤，程式可能直接不能執行。

### 2. 把 `=` 和 `==` 搞混

* `=` 是指定值
* `==` 是比較是否相等

這是非常常見的初學錯誤。

### 3. 以為 `input()` 讀進來就是數字

其實 `input()` 預設會回傳字串。
如果要拿來做數字運算，通常要自己轉型。

### 4. 看得懂範例，但自己寫不出來

這是最常見也最危險的情況。
所以每個小主題都應該：

* 自己重打一遍
* 自己改幾個值試試看
* 自己寫一個很小的變形版
