# Conditionals

## 這個資料夾在學什麼

這一課要學的是 Python 的條件判斷，重點是理解程式如何根據不同情況走不同路徑。

你會學到：

- `if`
- `elif`
- `else`
- 比較運算子的使用
- 如何用條件控制程式流程

---

## 程式重點

1. `if`
	- 當條件成立時，執行指定程式碼。
2. `elif`
	- 當前面的條件不成立時，再檢查下一個條件。
3. `else`
	- 當前面條件都不成立時，執行這一段。
4. 比較運算子
	- 例如 `>`, `<`, `>=`, `<=`, `==`, `!=`

---

## 預期你會看到

- 輸入不同分數，會得到不同等級
- `score >= 90`、`score >= 80` 這類條件會影響結果
- 程式會根據輸入值，走不同的判斷分支

## 常見錯誤

- 忘記縮排
- 把 `=` 和 `==` 搞混
- 條件寫錯順序
- `if`、`elif`、`else` 的區塊沒有對齊
