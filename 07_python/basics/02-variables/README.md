# Variables

## 這個範例在學什麼

這一課要建立 Python 最核心的觀念：變數。

你會學到：

- 如何把資料存進變數
- 四種常見型態：`str`、`int`、`float`、`bool`
- 如何用 `print()` 檢查變數內容
- 如何用 `type()` 觀察資料型態

---

## 檔案

- `main.py`

---

## 如何執行

在 `02-variables` 資料夾開終端機後執行：

```bash
python main.py
```

---

## 程式重點

1. 建立變數：
	- `name = "Andy"`
	- `age = 18`
	- `height = 172.5`
	- `is_student = True`
2. 用 `print()` 印出變數。
3. 做簡單運算：`next_year_age = age + 1`
4. 用 `type()` 看型態。

---

## 預期你會看到

- 變數值正確印出
- 明年年齡多 1
- 型態顯示為 `str`、`int`、`float`、`bool`

---

## 小練習

- 把 `name` 換成你的名字
- 把 `age` 換成你的年齡，檢查明年年齡
- 新增一個變數 `major`（科系）並印出
- 試著把 `age` 改成字串，觀察 `type(age)` 的變化

---

## 常見錯誤

- 變數名稱打錯（例如 `ag` 和 `age`）
- 字串忘記加引號
- 布林值大小寫寫錯（要 `True` / `False`）
