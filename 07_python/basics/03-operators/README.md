# Operators

## 這個範例在學什麼

這一課要學的是 Python 最常用的運算子，包含：

- 算術運算子
- 比較運算子
- 邏輯運算子
- 指派運算子

學完這一課，你會更清楚程式如何計算、比較與判斷。

---

## 檔案

- `operator.py`

---

## 如何執行

在 `03-operators` 資料夾開終端機後執行：

```bash
python operator.py
```

---

## 程式重點

1. 算術運算子
	- `+`, `-`, `*`, `/`, `//`, `%`, `**`
2. 比較運算子
	- `==`, `!=`, `>`, `<`, `>=`, `<=`
3. 邏輯運算子
	- `and`, `or`, `not`
4. 指派運算子
	- `=`, `+=`, `-=`, `*=`, `/=`

---

## 預期你會看到

- `a = 10`, `b = 3` 的運算結果
- 分數比較後的 True/False
- 條件組合判斷結果
- `x` 透過 `+=`, `*=`, `-=` 的更新結果

---

## 小練習

- 把 `a`, `b` 改成不同數字，觀察 `/` 和 `//` 的差別
- 新增一個條件：`score >= 80 and score < 90`
- 自己設計一個 `x` 的更新流程，例如 `x += 10`, `x //= 3`

---

## 常見錯誤

- 把 `=`（指定）和 `==`（比較）搞混
- `%` 取餘數意義不清楚
- 不了解 `and` 和 `or` 的條件組合邏輯
