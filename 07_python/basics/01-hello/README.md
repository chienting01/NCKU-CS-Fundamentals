# Hello World

## 這個範例在學什麼

這個範例是 Python 的第一個入門程式。  
目標是先熟悉三件事：

- 如何建立 .py 檔
- 如何執行 Python 程式
- 如何使用 print() 輸出文字

---

## 核心概念

### 1. Python 程式可以直接執行
和 C 語言不同，Python 通常不需要先編譯，直接執行即可。

### 2. print() 是最基本的輸出
print() 會把內容顯示在終端機。  
初學時可用它確認程式有正常執行。

---

## 如何執行

在 01-hello 資料夾開終端機後執行
---

## 預期輸出

Hello, Python!
我是 Python 新手。
這是我的第一個程式。

---

## 小練習

- 把第二行改成你的名字
- 新增一行，輸出你今天的學習目標
- 重跑一次，確認輸出有變化

---

## 常見錯誤

- 檔名打錯：hello.py 寫成 helo.py
- 目前資料夾不對：不在 01-hello 就直接執行
- 指令名稱差異：有些環境要用 python3
