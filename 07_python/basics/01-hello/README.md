# Hello World

## 這個範例在學什麼

這個範例是 Python 的第一個入門程式。  
目標不是做複雜功能，而是先熟悉：

- 如何建立 `.py` 檔
- 如何執行 Python 程式
- 如何使用 `print()`
- Python 程式最基本的執行方式

這一課的重點很小，但很重要。  
因為它代表你已經開始從「看語法」進入「自己執行程式」。

## 核心概念

### 1. Python 程式可以直接執行
和 C 語言不同，Python 不需要先編譯成可執行檔。  
通常可以直接用：

```bash
python hello.py
