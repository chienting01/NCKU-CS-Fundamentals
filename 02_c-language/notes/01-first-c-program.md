# First C Program

## What I learned
- A C program starts from `main`
- `printf` prints text to the terminal
- `\n` creates a new line
- C code must be compiled before running

## Program structure
1. Include header files
2. Write `main`
3. Put statements inside braces
4. Return `0`

## Common mistake
- Forgetting semicolon `;`
- Editing the `.c` file but not recompiling
- Trying to upload compiled file instead of source code
