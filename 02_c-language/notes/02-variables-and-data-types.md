# Variables and Data Types

## 為什麼需要資料型態

在 C 語言中，變數不是單純的「名字」，而是**記憶體中的一塊空間**。  
當程式建立一個變數時，電腦實際上需要在記憶體中保留一段空間來存放資料。

但問題是：電腦不能只知道「我要存東西」，還必須知道：

- 這塊空間要拿來存什麼類型的資料
- 這個資料需要多少大小
- 之後要用什麼方式讀取它
- 運算時要依照什麼規則處理它

這就是**資料型態（data type）**存在的原因。

例如在 C 裡常見的型態有：

- `int`：通常拿來存整數
- `float`：通常拿來存小數
- `char`：通常拿來存單一字元

這些型態不是隨便取的標籤，而是告訴編譯器與電腦：

> 這塊記憶體裡面的資料，應該用什麼方式被理解與操作。

也就是說，資料型態的作用不只是分類資料而已，還會影響：

- 記憶體配置
- 資料解讀方式
- 運算邏輯
- 輸入輸出格式

所以在 C 語言中，學會資料型態是非常基礎而且非常重要的一步。

## 什麼是強型別

C 語言屬於**強型別語言（strongly typed language）**。  
意思是：每個變數在宣告時，都要明確指定型別，程式會依照這個型別來處理資料。

例如：

```c
int age = 20;
float height = 172.5;
char grade = 'A';
