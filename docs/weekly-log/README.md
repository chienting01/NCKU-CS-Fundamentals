# Weekly Log

This folder stores weekly learning reviews.

Each weekly log should answer:
1. what was completed
2. what was learned
3. what problems appeared
4. what should be done next

Suggested file naming:
- `week-01.md`
- `week-02.md`
- `week-03.md`

Weekly logs should stay practical and concise.
They are used as execution evidence, not as polished essays.
